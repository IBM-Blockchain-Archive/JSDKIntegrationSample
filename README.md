# BMXHyperledgerFabricJSDKIntegrationSample

## Installing
 Run command:
  ```mvn install```
 
## Running

Run command: 
```mvn exec:java -Dexec.mainClass="org.cr22rc.BMXHyperledgerFabricJSDKIntegrationSample"```
